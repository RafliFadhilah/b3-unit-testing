Program Pengolahan Nilai Mahasiswa 
1.ÔÇï Buat project sederhana (Pengolahan nilai mahasiswa: validasi, penentuan nilai akhir 
dan penentuan grade serta status kelulusan) menggunakan environment yang sudah 
dibuat dengan Main App:  
ÔùÅÔÇï User menginputkan nilai mahasiswa berulang kali 
ÔùÅÔÇï Program memvalidasi input 
ÔùÅÔÇï Program menghitung nilai akhir 
ÔùÅÔÇï Program menentukan grade & kelulusan dan menampilkannya kelayar 
ÔùÅÔÇï Program mengelola proses secara berulang (loop menu) sehingga user dapat 
berinteraksi secara berulang 
2.ÔÇï Buat modul program yang terisolasi dari modul lain. Modul ini digunakan untuk 
melakukan proses pengolahan nilai, yaitu buah modul input data, validasi data, dan 
modul komputasi untuk menentukan nilai Akhir, grade dan status kelulusan. 
Pembuatan modul harus dibuat modular dan terisonalasi. Setiap modul dipanggil 
dalam sebuah modul utama pemrosesan pengolahan nilai dan dikembalikan nilainya 
ke main program sebagai interface aplikasi. 
3.ÔÇï Modul validasi data 
a.ÔÇï input nilai_tugas, nilai_uts, nilai_uas 
b.ÔÇï Output Valid (True) / Tidak valid (False) 
c.ÔÇï Proses : tentukan kevalidan data berdasarkan aturan: 
1. Jika semua nilai antara 0ÔÇô100 ÔåÆ valid 
2. Jika ada nilai < 0 atau > 100 ÔåÆ tidak valid 
3. Jika semua nilai = 0 ÔåÆ tidak valid (anggap belum input) 
d.ÔÇï Modul validasi ini dipanggil oleh modul input data, selama modul menghasilkab 
tidak valid, maka user diminta untuk menginputkan data sampai benar

4.ÔÇï Modul Perhitungan Nilai Akhir 
a.ÔÇï input nilai_tugas (30%), nilai_uts (30%), nilai_uas (40%) 
b.ÔÇï Output nilai_akhir 
c.ÔÇï Proses : tentukan perhitungan nilai akhir berdasarkan aturan: 
1. nilai_akhir = (0.3 * tugas) + (0.3 * uts) + (0.4 * uas) 
2. Jika nilai tidak valid ÔåÆ return error (-1) 
3. Jika nilai akhir > 100 ÔåÆ dianggap error 
5.ÔÇï Modul Penentuan Grade

2. Jika 70ÔÇô84 ÔåÆ B 
3. Jika 60ÔÇô69 ÔåÆ C 
4. Jika 50ÔÇô59 ÔåÆ D 
5. Jika < 50 ÔåÆ E 
6.ÔÇï Modul Penentuan Kelulusan
